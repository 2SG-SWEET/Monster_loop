# Monster Loop - 克苏鲁风格宠物精灵设计文档

> 中世纪像素风怪物伙伴设计规格

---

## 1. 整体风格定义

### 1.1 视觉风格
- **画风**: 中世纪手稿插画 + 克苏鲁恐怖元素
- **像素规格**: 32x32 或 48x48 像素
- **色彩**: 暗色调为主，使用深绿、暗紫、铁锈红、腐尸绿
- **线条**: 粗黑边线，内部细节用2-3色渐变
- **参考**: 暗黑地牢(Darkest Dungeon) + 克苏鲁的呼唤

### 1.2 设计原则
- 每个怪物应有"可爱但诡异"的平衡
- 融入触手、眼球、粘液等克苏鲁标志性元素
- 中世纪元素：锁链、蜡烛、羊皮纸、炼金瓶、宗教符号

### 2.1 【眼魔侍从】Eye Spawn

#### 基础信息
- **稀有度**: ★☆☆☆☆ (普通)
- **属性**: 精神/感知
- **定位**: 辅助型

#### 外观描述
```
体型: 漂浮的球状生物，直径约30cm
主体: 暗紫色肉球，表面有血管纹理
核心特征:
  - 中央一只巨大独眼，黄绿色虹膜，竖瞳
  - 眼睛周围有3-4只小型副眼，随机分布
  - 底部垂下3条细小触手，末端有小吸盘
中世纪元素:
  - 头顶戴着破损的修士兜帽
  - 脖子上挂着生锈的小铃铛
  - 瞳孔中隐约可见古老符文

像素细节:
  - 主眼占身体1/3大小
  - 副眼用2-3像素点表现
  - 触手用3-4像素宽的曲线
  - 兜帽用深棕色，边缘破损效果
```

### 2.2 【触手猫】Tentacle Mew

#### 基础信息
- **稀有度**: ★★☆☆☆ (常见)
- **属性**: 敏捷/暗影
- **定位**: 刺客型

#### 外观描述
```
体型: 家猫大小，约40cm长
主体: 黑色短毛，但毛发呈现不自然的油光
核心特征:
  - 尾巴是一条完整的触手，末端有眼状花纹
  - 背部有2-3条小触手可以竖起
```

### 2.3 【脓液史莱姆】Ichor Slime

#### 基础信息
- **稀有度**: ★☆☆☆☆ (普通)
- **属性**: 毒/腐蚀
- **定位**: 坦克型

#### 外观描述
```
体型: 不规则 blob，直径约50cm
主体: 黄绿色半透明粘液，内部有气泡
核心特征:
  - 身体内部漂浮着各种残骸（骨头、牙齿、小饰品）
  - 表面不断产生气泡然后破裂
  - 偶尔浮现出痛苦的人脸轮廓然后消散
  - 底部有假装的"脚"，其实是凝固的粘液
中世纪元素:
  - 内部有一枚生锈的骑士徽章
  - 半消化状态的羊皮纸卷轴
  - 一小截断裂的锁链

像素细节:
  - 用不规则圆形表现，32x32像素
  - 内部残骸用2-3像素点表现
  - 气泡用1-2像素的圆形，有透明感
```

## 3. 通用设计规范

### 3.1 像素艺术技法

#### 抗锯齿
- 使用1-2像素的过渡色
- 避免纯黑纯白，用深色/浅色替代

#### 轮廓线
- 外轮廓用2-3像素的深色线
- 内部细节用1像素线
- 轮廓线颜色根据环境光变化

#### 光影
- 光源统一从左上方45度
- 使用2-3层阴影（核心阴影、投影、环境光遮蔽）
- 高光用1-2像素点表现

### 3.2 动画建议（如后续需要）

虽然当前只需要静态图，但建议按以下方式绘制以便后续动画化：
- 分层绘制（身体、装备、特效分离）
- 关键部位单独保存（眼睛、触手等）
- 预留变形空间

### 3.3 导出规格

| 项目 | 规格 |
|------|------|
| 分辨率 | 32x32 / 48x48 / 64x64 |
| 格式 | PNG (透明背景) |
| 色彩模式 | RGBA |
| 命名 | Pet_[英文名]_Idle.png |

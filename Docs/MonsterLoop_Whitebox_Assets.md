# Monster Loop - 白模场景资源设计文档

> Loop Hero 风格基础跑道白模设计

---

## 1. 整体视觉风格

### 1.1 参考风格
- **参考游戏**: Loop Hero (循环英雄)
- **视觉特点**: 
  - 暗黑像素风
  - 高对比度黑白灰
  - 简洁的几何形状
  - 复古CRT显示器效果

### 1.2 白模设计原则
- 使用纯色块代替精细纹理
- 黑色背景 + 白色/灰色元素
- 清晰的轮廓线
- 功能性优先，美观其次

---

## 2. 场景元素规格

### 2.1 基础跑道 (Loop Path)

#### 视觉设计
```
形状: 完美的圆形跑道
颜色: 深灰色 (#4A4A4A) 路径，黑色 (#1A1A1A) 背景
宽度: 40像素
样式: 纯色填充，无纹理

布局:
- 圆心: 屏幕中心
- 半径: 250像素 (500x500像素区域)
- 路径上均匀分布10个格子位置
```

#### 技术规格
- **节点类型**: Path2D + Line2D (或 Polygon2D)
- **路径点**: 60个点构成平滑圆形
- **碰撞**: 无需碰撞，仅视觉

---

### 2.2 格子位置 (Grid Slots)

#### 视觉设计
```
形状: 圆形标记点
颜色: 白色 (#FFFFFF) 边框，透明填充
大小: 20像素直径
样式: 2像素宽度的空心圆

状态显示:
- 空格子: 白色空心圆
- 已放置: 填充对应模块颜色
- 可放置: 闪烁白色 (动画)
```

#### 技术规格
- **节点类型**: Marker2D + 可视化子节点
- **数量**: 10个，均匀分布在跑道上
- **间距**: 36度 (360/10)

---

### 2.3 玩家角色 (Player)

#### 视觉设计
```
形状: 简单的三角形箭头
颜色: 亮白色 (#FFFFFF)
大小: 16x16像素
朝向: 指向移动方向

样式:
    ▲
   ███
  █████
```

#### 技术规格
- **节点类型**: CharacterBody2D
- **子节点**: 
  - Polygon2D (三角形)
  - 或 Sprite2D (简单图标)
- **动画**: 轻微上下浮动 (呼吸效果)

## 3. 场景节点树结构

```
GameScene (Node2D)
├── Background (ColorRect)
│   └── Color: #0A0A0A (近黑色)
│
├── Camera2D
│   ├── Position: (0, 0)
│   ├── Zoom: 1.0
│   └── Anchor Mode: Drag Center
│
├── PathContainer (Node2D)
│   ├── LoopPath (Path2D)
│   │   └── Curve2D: 圆形路径
│   │
│   ├── PathVisual (Line2D)
│   │   ├── Width: 40
│   │   ├── Color: #4A4A4A
│   │   └── Points: 跟随Path2D
│   │
│   └── GridMarkers (Node2D)
│       ├── GridSlot_0 (Marker2D)
│       │   └── Visual (Line2D 圆形)
│       ├── GridSlot_1 (Marker2D)
│       │   └── Visual (Line2D 圆形)
│       ├── ... (共10个)
│       └── GridSlot_9 (Marker2D)
│           └── Visual (Line2D 圆形)
│
├── PlayerContainer (Node2D)
│   └── Player (PathFollow2D)
│       └── Visual (Polygon2D 三角形)
│
├── TileContainer (Node2D)
│   └── [放置的地块模块实例]
│
├── BossSpawnPoint (Marker2D)
│   └── Visual (Polygon2D 骷髅形状)
│
└── UI (CanvasLayer)
    ├── TopBar (HBoxContainer)
    │   ├── LoopCount (Label)
    │   └── BossProgress (ProgressBar)
    │
    └── HandUI (Control)
        └── CardContainer (HBoxContainer)
```

## 5. 动画规格

### 5.1 玩家移动
- **类型**: 沿路径循环移动
- **速度**: 100像素/秒
- **朝向**: 始终面向移动方向
- **动画**: 轻微上下浮动 (±2像素，1秒周期)

### 5.2 格子高亮
- **触发**: 鼠标悬停或选择手牌
- **效果**: 白色圆环放大 + 透明度变化
- **时长**: 0.3秒淡入淡出

### 5.3 地块放置
- **效果**: 从0缩放到1 + 淡入
- **时长**: 0.2秒
- **缓动**: Back Out

### 5.4 BOSS生成
- **效果**: 红色脉冲 + 屏幕轻微震动
- **时长**: 1秒
- **缓动**: Elastic Out
